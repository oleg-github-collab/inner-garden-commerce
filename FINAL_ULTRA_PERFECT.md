### 🌍 **ЛОКАЛІЗАЦІЯ - МИТТЄВА:**

#### **Event-Driven Architecture:**
```javascript
// Preloader відправляє
window.dispatchEvent(new CustomEvent('languageSelected', {
  detail: { language: 'en' }
}));

// I18n система ловить
window.addEventListener('languageSelected', (event) => {
  this.setLanguage(event.detail.language); // Миттєво застосовує
});
```
